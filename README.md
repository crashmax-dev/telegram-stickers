## Running Locally

```bash
$ git clone https://github.com/crashmax-dev/telegram-stickers.git
$ cd telegram-stickers
$ npm install
$ npm run dev
```

Create a `.env` file similar to [`.env.example`](https://github.com/crashmax-dev/telegram-stickers/blob/master/.env.example).
